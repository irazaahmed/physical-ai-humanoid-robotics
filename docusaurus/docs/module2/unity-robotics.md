# Unity Robotics Hub

## Learning Objective
Understand Unity as a robotics simulation platform for AI applications.

## Key Concepts
- Unity Robotics Hub and its components
- URDF importer for robot models
- Physics simulation in Unity
- Visual rendering and HMI interfaces
- Integration with ML-Agents for AI training

## Lab/Practical
Import a robot model into Unity and create a basic simulation scene.
