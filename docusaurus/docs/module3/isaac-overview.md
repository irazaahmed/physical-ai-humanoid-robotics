# NVIDIA Isaac Platform Overview

## Learning Objective
Learn the architecture and components of NVIDIA Isaac as an AI-powered robotics platform.

## Key Concepts
- Isaac platform architecture and components
- Isaac Sim for GPU-accelerated simulation
- Isaac ROS for ROS integration
- Isaac Apps for reference implementations
- Isaac SDK for custom development
