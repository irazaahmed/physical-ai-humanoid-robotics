# Lab 2: AI Perception in Robotics

## Learning Objective
Implement AI-based object detection and tracking in a simulated robotic environment.

## Key Concepts
- Deep learning model integration in robotics
- Real-time object detection and classification
- Tracking algorithms for dynamic objects
- Performance optimization for robotics
- Accuracy vs. speed trade-offs

## Lab/Practical
Develop an AI perception system that detects and tracks objects in Isaac Sim environment.
