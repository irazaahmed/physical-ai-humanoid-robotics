# Lab 1: Isaac Sim Fundamentals

## Learning Objective
Build and test a simple robot simulation in Isaac Sim with AI capabilities.

## Key Concepts
- Isaac Sim environment setup
- Robot model integration in simulation
- GPU-accelerated physics simulation
- AI perception and control integration
- Simulation-to-reality transfer

## Lab/Practical
Create a differential drive robot in Isaac Sim, add AI-based sensors, and implement basic navigation.
